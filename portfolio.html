    GeoExpanse

    A Graphical User Interface based application made in Python to model geometric structures including Minimal Surfaces, Regular Surfaces, and a variety of Polyhedra and Solids. The goal of this project is to provide an educational outlet for students to explore un-familiar geometric structures.

    VPython-Based GUI

    A VPython compatible Tkinter framework for computational and visual models. Current model for the framework is a orbital 2-body system. The goal of this project is to create a framework for Tkinter and VPython to work together. The applications include a more interactive approach to visual and computational modeling
