    GeoExpanse

    A Graphical User Interface based application made in Python to model geometric structures including Minimal Surfaces, Regular Surfaces, and a variety of Polyhedra and Solids. The goal of this project is to provide an educational outlet for students to explore un-familiar geometric structures.

    Visual Python-Based GUI

    A Visual Python compatible Tkinter framework for computational and visual models. Current model for the framework is a orbital 2-body system. The goal of this project is to create a framework for Tkinter and VPython to work together. The applications include a more interactive approach to visual and computational modeling

    Surface-Climate Ambiguity in Radio Frequency Modeling

    A research endeavor with the purpose of developing a generalized model for radio propagation networks. The eventual goal is to have the capabilities to model networks on other-worldly surfaces such as Mars and Titan. The final product is expected to be a Python Tkinter application that can import and export various types of astrophysical and atmospheric data for analysis.
